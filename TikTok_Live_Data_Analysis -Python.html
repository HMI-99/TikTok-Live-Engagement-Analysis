Step 6: Calculate Week-on-Week Percentage Growth in Views¶

To understand how my TikTok Live views are changing week by week, I calculate the percentage growth between each week.

I use the pandas .pct_change() method, which compares each row to the previous one and gives the relative change as a decimal. Multiplying by 100 converts it to a percentage.

This helps me identify trends — whether views are increasing, decreasing, or holding steady over time.

In [9]:
# make sure data is sorted by date
df = df.sort_values("Date Posted")

# Calculate percentage growth of Views compared to previous week
df['Views_Pct_Growth'] = df['Views'].pct_change()*100

# Preview the result (first 10 rows)

Step 7: Visualize TikTok Views Over Time¶

Next, I create a line chart showing how the number of views changes over time.

The X-axis represents the dates of my TikTok Lives, and the Y-axis shows the corresponding views for each live.

This visual allows me to quickly spot patterns, spikes, or drops in engagement, which I can investigate further.

In [10]:
import matplotlib.pyplot as plt  # make sure this import is at the top of your notebook

plt.figure(figsize=(12,6))
plt.plot(df['Date Posted'], df['Views'], marker='o', linestyle='-')
plt.title('TikTok Views Over Time')
plt.xlabel('Date')
plt.ylabel('Number of Views')
plt.grid(True)
plt.xticks(rotation=45)
plt.tight_layout()
plt.savefig('views_over_time.png')
plt.show()
